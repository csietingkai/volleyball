想要 log 在 console 跟檔案那邊輸出
終端機輸出最低等級為 INFO
檔案輸出最低等級為 TRACE

輸出的格式長這樣
========================================
[現在時間] [Log等級] 		(class名稱) 輸出訊息
[TimeStamp] [%Severity%] 	(class name) message
========================================

可以參考 https://blog.scalyr.com/2018/07/getting-started-quickly-c-logging/ 這篇
class名稱前面要對齊
還有 log 檔的名稱會叫 '[今天的日期]_[隨機六位英文數字].log'，比如說 '2018-10-20_3jdl8s.log'這樣

然後寫出來的 code 要符合格式
像是
const int function_name(const int num)
{
	int num_temp = num;
	int some_variable;
	// some implements
}

這樣
1. 回傳的資料型態前面盡量加 const (void 除外)
2. 大括號的開頭放在下一行
3. 變數名稱請務必要讓人看得懂，單字之間用底線（'_'）分開
